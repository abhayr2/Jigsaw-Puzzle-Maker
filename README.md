# Jigsaw Puzzle Maker

A simple web-based jigsaw puzzle game built with vanilla HTML, CSS, and JavaScript.  
Upload your own image, select a difficulty, and instantly generate a playable drag-and-drop puzzle.

---

## Features

- **Custom Images:** Upload any image from your device to use as the puzzle.  
- **Selectable Difficulty:** Choose from multiple difficulty levels, from 5 pieces (Very Easy) to 100 pieces (Expert).  
- **Dynamic Grid Generation:** The puzzle board and pieces are dynamically created based on the chosen difficulty.  
- **Drag & Drop:** A clean, intuitive drag-and-drop interface for moving pieces from the "pieces" container to the puzzle board.  
- **Win Detection:** The game automatically detects when all pieces are in their correct slots and notifies you of completion.

---

## How to Use

1. Open `index.html` in your web browser.  
2. Click the **"Choose File"** button to upload an image.  
3. Select a difficulty from the dropdown menu.  
4. Click **"Create Puzzle"**.  
5. Drag the shuffled pieces from the right-hand container onto the dashed grid on the left to solve!

---

## Technologies Used

- **HTML5:** For the basic structure of the app.  
- **CSS3:** For all styling, including the grid layout and visual feedback.  
- **JavaScript (ES6+):** For all application logic, including file handling, dynamic element creation, and the complete drag-and-drop functionality.
